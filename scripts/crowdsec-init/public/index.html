Quickstart

  1. 1

    Check LAPI Health

    Start by confirming the LAPI Health card is green. If it shows offline, wait a few seconds and check that your CrowdSec container is running before changing settings.

  2. 2

    Set the LAPI Key

    Use “Generate New Key” for the safest default, or paste your own key if you already have one. A valid key unlocks secure communication between Traefik and CrowdSec.

  3. 3

    Register the Bouncer

    Keep the default name `traefik-bouncer` unless you need a custom name for your environment. Click Register, then confirm the Bouncer card changes to a healthy state.

  4. 4

    Enroll in CrowdSec Console (Optional)

    Paste an enrollment key from app.crowdsec.net if you want centralized visibility and management. You can skip this step and come back later without breaking local protection.

  5. 5

    Install Collections & Finish

    Install the collections relevant to your stack so detections match your services. When the status cards are healthy, use “Finalize Now” or let auto-exit complete setup.
