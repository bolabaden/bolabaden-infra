🆘 TROUBLESHOOTING:

    Problem: "secret file not found"
    Fix: ./scripts/generate-secrets.sh -f

    Problem: "permission denied"  
    Fix: chmod 700 secrets/ && chmod 600 secrets/*.txt

    Problem: Service won't start
    Fix: docker compose logs <service> | grep -i secret

    Problem: Empty secret file
    Fix: Edit .secrets, then ./scripts/generate-secrets.sh -f
